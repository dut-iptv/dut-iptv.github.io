## plugin.video.f1tv

### About

-Watch F1 TV (live) content

### Features

-Unofficial 3rd Party F1 TV plugin for Kodi

### Maximum supported Resolution

- 1080P (50hz)

### Required

-Subscription to F1 TV (not free)

-Kodi 19 or higher with Widevine Support (free)
